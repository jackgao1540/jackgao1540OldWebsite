Segment Tree Lazy Propagation Visualization




This program will generate a segment tree with the given values. The user will be able to update and query the segment tree. It uses lazy propagation in order to make updates and queries more efficient.


Tesla stock price prediction

Using long short-term memory neural networks and the python library Keras to predict changes in the TSLA stock price. The model takes the last 60 days of close prices as input and outputs a prediction of how the stock's close price will change in the following day. The result of training this network was a Keras model that learned to predict changes in the TSLA close price with a root mean square error of 0.170, which is pretty accurate.

What type of breast cell is this?

Using convolutional neural networks and Tensorflow, this python program is trained to differentiate between three types of breast cells: MCF-7, MCF-10A, and MDA-MB-231, the latter two being cancerous.
The training dataset was obtained from this article.
The steps taken to create this program, as well as the configuration of the convolutional neural network model made in keras, were adapted from the procedure described in this article. The program was made for the "Group 4" project in the IB year 1 curriculum.

Minesweeper - Java string game

A project I made in grade 10 for my string game project, uses ASCII characters to produce crude graphics for a minesweeper game. Please ignore any unusual or odd jokes that were programmed into the game, as it was made some time ago.

F indicates a player-flagged square
